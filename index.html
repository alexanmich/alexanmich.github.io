Data Science Project with EDA
GitHub Repo

The aim of this project was to estimate the effect of COVID-19 on New Zealand spending. By predicting how much kiwis would have spent using data prior to lock-down, I could estimate how much less kiwis were spending. The language R was used to perform the data analysis

Playing Card Simulation
GitHub Repo

Over the past four years I have collected playing cards I found on the street. The goal is to find a unique 54 card deck. I want to know how many cards I should expect to pick up to find a full unique deck.

I used R to simulate multiple runs of picking up random cards until a full unique (54) deck is found. I used the numbers from these runs to get an expected number of cards to find.
